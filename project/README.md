Клиен/Сервер на io
Комманды :
-auth [логин] [пароль]                              Авторизация на сервере (Пример: -auth n1@mail.com 1)
-l | -list                                          Выводит список файлов
-u | -upload                                        Загрузить файл на сервер
-d | -download [номер файла из списка]              Скачать файл
-exit                                               Отключить клиента
