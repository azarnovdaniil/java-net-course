«Пример пути к файлу: pack\pack2\test.txt, к папке: pack\pack2\pack3»
«Если элемент в ц.хранилище: test.txt, pack3»
Команды:
создать файл:           ~cf (имя файла)
создать папку:          ~cd (имя директории)
переименовать элемент:  ~r  (имя элемента) (новое имя элемента)
удалить элемент:        ~d  (имя элемента)
